# Retail Sales Performance Dashboard (Excel)

## Tools Used
- Microsoft Excel

## Project Overview
This project presents an interactive Excel dashboard built to analyze retail sales data.
The dashboard highlights sales trends, customer demographics, order status, sales channels,
and regional performance using pivot tables, charts, and slicers.

## Dataset Description
The dataset contains transactional retail sales data including:
- Order details and order status
- Customer gender and age groups
- Product categories
- Sales channels (Amazon, Flipkart, etc.)
- Geographic information (state-level)

## Key Analysis Performed
- Monthly sales and order trends
- Sales distribution by gender and age group
- Top-performing sales channels
- Order status breakdown
- Top 5 states by sales revenue

## Key Insights
- Majority of revenue is driven by female customers
- Adult age group contributes the highest sales
- Amazon and Flipkart are the top-performing channels
- Maharashtra leads in overall sales among states

## Dashboard Interactivity
Slicers are provided to filter data by month, category, and channel for exploratory analysis.
Some visuals are designed as summary views to highlight overall performance trends.

## Outcome
This project demonstrates practical Excel data analysis skills, dashboard design,
and the ability to extract business insights from raw transactional data.
